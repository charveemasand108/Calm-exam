## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| UI | SwiftUI |
| State Management | ObservableObject + @Published |
| Persistence | UserDefaults + JSONEncoder/Decoder |
| Reactive | Combine |
| Language | Swift 5.9 |
| Min iOS | iOS 16.0+ |

---

## 🚀 Getting Started

1. Clone the repo
```bash
   git clone https://github.com/charveemasand108/Calm-exam.git
```
2. Open `Calm exam.xcodeproj` in Xcode
3. Select a simulator or your device
4. Hit **Run** ▶️
